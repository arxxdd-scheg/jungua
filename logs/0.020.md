# jungua - version 0.020
28.05.2026
## что нового?

- повторение всего что напишет пользователь удалено, для этого подразумевается другая команда.
- команда say() для вывода текста, который напишет юзер в скобках.
- команда iam для вывода username пользователя изменилась в названии на whoami
- команда rename для изменения текущего username на новый.
- команда clear или cls для очистки командной строки.
