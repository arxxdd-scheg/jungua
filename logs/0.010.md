# jungua - version 0.010
27.05.2026
## что нового?

- повторение всего что напишет пользователь.
- команда fetch для вывода иконки консоли, версии, общего места на диске, общего места на оператиивной памяти, а также времени, которое было во время выполнения команды.
- команда iam для вывода username пользователя, заданный при начале работы с jungua.
- команда exit для выключения консоли.
